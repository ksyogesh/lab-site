A Sneak peek at search engines

Around 1990, Tim Burners-Lee had a bad dream in which a scary monster with "HTTP" etched into its hide slowly ate up all of the Earth's resources. The main way people shared data back then was via FTP. If you had a file you wanted to share you would set up an FTP server. If someone was interested in retrieving the data they could do using an FTP client. This process worked effectively in small groups. But the data became as much fragmented as it was collected as the network swelled and there evolved a concept of a practical search engine. In early 90s, ARCHIE and VERONICA were the first to arrive primitive search engines. Today's search giants include:

 

  • YAHOO!: In April 1994 David Filo and Jerry Yang created Yahoo as a collection of their favorite web pages. Their initial URL was www.akebono.stanford.edu. As their number of links grew they had to be reorganized and thus it became a searchable directory.

 

  • Google: In 1998 the most powerful to date, Google was launched by two young and energetic youth Sergey Brin and Larry Page. This search engine was their final year project at Stanford University. Today, it has become a Web 2.0 giant providing various services.

 

  • Bing: Bing is a web decision engine and Microsoft's current incarnation of its search technology and replacement to Live search. Bing is referred by some tech gurus as a recursive acronym for "Bing Is Not Google".!

 

Cloud Computing

Cloud Computing is a technology that uses the internet and central remote servers to maintain data and applications. It allows consumers and businesses to use applications without installation and access their personal files at any computer with internet access. This technology allows for much more efficient computing by centralized storage, memory, processing and bandwidth.

Few public cloud services are:

Skydrive: SkyDrive is a File hosting service from Microsoft that allows users to upload files to a cloud storage and then access them from a Web browser. The service offers 25GB of whooping free personal storage, with individual files limited to 100 MB. The service is built using HTML5 technologies.

Google Docs: Google Docs is a free, Web-based office suite, and data storage service offered by Google. It allows users to create and edit documents online while collaborating in real-time with other users. Google is ready to launch Google Drive as their new cloud services which includes all its other services under one service.

Mozy:  Mozy is an online back up cloud service from EMC corporation(EMC2). It provides 2GB of free storage.

About

          Computer Science VII Semester has two labs according to VTU syllabus. The subject code of networks lab is 06CSL77 and web lab is 06CSL78. They are web programming lab and networks lab. Web programming lab includes programs on JavaScript, XHTML, CSS, Perl, PHP and so on. These programs gives example of how a web page is created and how they are executed in the browser. The networks lab deals with simulating different types of traffic in the network using different protocols and LAN connection. There are some programs which also illustrates with algorithms which comes across the networking protocols.

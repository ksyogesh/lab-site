Labs

As prescribed by VTU, there are two labs. They are:

1.  Web Programming Lab(06CSL78)

2.  Networks Lab

We execute our lab exercises in Local Area Network connected through a Apache server in Linux environment using ssh(secure shell) login.

IEEE 802 Network Standards:

IEEE 802.1        Standards related to network management.

IEEE 802.2        Standards related to Logical Link Control(LLC)

IEEE 802.3        Standards for Ethernet or LAN

IEEE 802.4        Standards for token bus networks(disbanded)

IEEE 802.5        Standards for token ring networks

IEEE 802.6        Standards for MAN(disbanded)

IEEE 802.7        Standards for LAN using Coaxial Cable

IEEE 802.8        Standard for fiber optic TAG(disbanded)

IEEE 802.9        Standard for IsoEnet or Isochronous LAN

IEEE 802.10      Standard for Interoperable LAN/MAN Security

IEEE 802.11      Standard for Wireless LAN(WLAN)

IEEE 802.12      Standard for Gigabit Ethernet

IEEE 802.13      Not Used !

IEEE 802.14      Standard for Cable Modems

IEEE 802.15      Standard for Wireless PAN

 

 

 

Port Numbers

Few well known port numbers are:

21: File Transfer Protocol (FTP)

22: Secure Shell (SSH)

23: Telnet remote login service

25: Simple Mail Transfer Protocol(SMTP)

53: Domain Name System (DNS) service

80: Hypertext Transfer Protocol (HTTP)

110: Post Office Protocol (POP)

119: Network News Transfer Protocol

143: Internet Message Access Protocol

161: Simple Network Management Protocol

443: HTTP Secure
